# Facebook-API-Params-Generator
Generate Facebook api parameters. 生成Facebook接口参数。

# Usage
## dyn.js
```JavaScript
// Original parameters
createDyn()

// Random parameters
createDyn(1)
```
## csr.js
```JavaScript
// Original parameters
createCsr()

// Random parameters
createCsr(1)
```
